# Prerequisites
Before you get started with developing with the Noahh SDK, there are a few prerequisites that you should know about!
* Prior C++ knowledge is **HIGHLY** recommended. We do not intend in helping you at C++ from the very basics.
* Installing [Noahh](https://noahh-sdk.org/install/) itself! you want to test your own mods dont you?

It is recommended that you follow the tutorials on this chapter in order.
